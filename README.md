# 📚 Bookup

Sistema web para gerenciamento de biblioteca pessoal, desenvolvido com foco em qualidade de software, testes automatizados e integração contínua.

---

# 🚀 Objetivo do Projeto

O Bookup foi desenvolvido para permitir que usuários realizem o gerenciamento de seus livros pessoais, incluindo:

- cadastro de livros
- edição de livros
- exclusão de livros
- autenticação de usuários
- controle de status de leitura

O projeto também possui foco em práticas de qualidade de software, incluindo testes automatizados, cobertura de testes e integração contínua.
